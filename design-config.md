# Marketplace Design Standards

## Color Palette (Digital Agrarian)
- **Primary**: #0d631b (Forest Green)
- **Primary Container**: #2e7d32
- **Surface**: #f4faff (Cloud Blue)
- **Surface Container Low**: #e6f6ff
- **Outline**: #d9e5ec

## Typography
- **Display**: 'Manrope', sans-serif (800 for headers)
- **Body**: 'Public Sans', sans-serif (400 for text, 600 for labels)

## Component Constraints
- **Empty State Icons**: Always wrap in `.mp-empty__icon` and constrain SVG to `40px` x `40px`.
- **Navbar Height**: 68px.
- **Product Card Aspect Ratio**: 150px height for visual container.
- **Hero Banner Padding**: 1.25rem vertical minimum.

## Interaction States
- **Hover**: 0.18s transition, subtle transform translateY(-2px).
- **Active Chips**: Forest Green background with White text.
